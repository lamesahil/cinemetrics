# CineMetrics 🎬
A full-stack, data-driven personal film logging and analytics dashboard. 

### 🚀 Live Demo
https://cinemetrics-zeta.vercel.app/auth.html

### 🛠️ Tech Stack
* **Frontend:** HTML, Tailwind CSS, JavaScript (Vanilla/DOM Manipulation)
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Authentication:** JSON Web Tokens (JWT) & bcryptjs
* **External API:** TMDB (The Movie Database) API
* **Deployment:** Vercel (Frontend) & Render (Backend)

### ✨ Features
* Secure user authentication (Sign Up / Sign In).
* Automated movie metadata fetching (Release Year, Exact Genres) via TMDB API.
* Dynamic UI for marking films as "Watched" or "Pending".
* Real-time personal analytics (Total Films, Average Rating, Top Genre).
